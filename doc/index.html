o

What is this?

o is a full text search engine. o uses N-gram method. o handles Unicode (UTF-8).

About name

o is pronounced as "shaling". "shaling" is a Japanese, and means a "wheel". o's shape is a wheel, so this is "shaling".

Requirements

  • x86
  • Ubuntu Linux 8.04
  • libz.so.1
  • libbz2.so.1.0
  • librt.so.1
  • libpthread.so.0
  • libm.so.6
  • libc.so.6
