Usage

o provides a command line interface that name is o. You can handle a o's database with this tool.

Create a database

You can create new database by the following command.

$ o create db

The above command makes a directory of "db" which includes indexes.

Register a documentation

$ o put db < foo

The above command registers a documentation in the file "foo" to the index "db". Charactor encoding of contents in documentations must be UTF-8.

Search documentations

The following command searches "foo" in the index "db".

$ o search db foo

If documentations are found, o outputs IDs of these documentations. You can get their contents with these IDs by the "get" command.

Get a documentation

$ o get db 42

The above command outputs contents of the documentation which ID is 42.
